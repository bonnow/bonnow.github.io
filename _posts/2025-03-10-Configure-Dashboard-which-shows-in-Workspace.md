When you create the dashboard which shows in the Workspace, we need to create and configure it in the certain place.<br/>
> Workspace에 보여지는 dashboard를 구성할때는 정해진 곳으로 가서 구성을 해야합니다.

To create or change the dashboard which shows in the workspace.
Navigate to "Platform Analytics > Library > Dashboards". <br/>
> Workspace에서 보여지는 dashboard를 만들려면, "Platform > Library > Dashboards" 메뉴로 가세요.
